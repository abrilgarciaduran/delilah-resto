Delilah Resto
3er proyecto de la carrera de Desarrollo Web Full Stack en ACAMICA
El proyecto consiste en en una Rest API para manejar los pedidos, productos y
usuarios de un Resto

Tecnologías utilizadas:
Node.js
Nodemon
Express
JWT (Json Web Token)
MySQL
Sequelize
Postman
Swagger

Documentación de la API
Encuentre la dcumentación en el archivo docs.yaml, puede copiarla en Swagger
o importarla. Allí encontrará los endpoints y la información necesaria para usar cada uno.
